# register_contract

register_contract 命令把本地编译好的智能合约字节码注册到 HX 链上。

> 请求
```
{
  "jsonrpc": "2.0", 
  "method": "register_account", 
  "params": ["test", "0.00001", "40", "d:\\HXWallet\\randombonus.gpc"], 
  "id": 1
}
```

> 输入参数

* 发起注册合约交易的账户名
* gas 价格，即执行每一步花费的 HX 数量，最少是 0.00001
* gas 限制，最大执行步骤，实际超过这个数会执行失败，返回错误。
* 合约字节码文件的路径
