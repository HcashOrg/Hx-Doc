# get_first_contract_address

get_first_contract_address 命令获取第一个合约地址。

> 请求
```
{
  "jsonrpc": "2.0", 
  "method": "get_first_contract_address", 
  "params": [""], 
  "id": 1
}
```
> 响应

```
{
    "id": 1,
    "jsonrpc": "2.0",
    "result": "HXCGzXcFfPDwd1BHQj8VzBS9roLjkCdaGwUN"
}
```

> 返回参数

- **result**: 返回合约地址
